## CCPN NEF Utilities

this directory contains a series of modules for the processing of Nef files.

  1. *CompareNef.py* - a series of routines to compare the contents of two Nef files.
               
  2. *GenericStarParser.py* - a Star-type file parser
  3. *Specification.py* - contains the code for handling the NEF specification.
  4. *StarIo.py* - contains the code for handling the I/O for NEF and NmrStar formats.
  5. *StarTokeniser.py* - STAR file tokenizer
  6. *NefImporter.py* - a series of routines for reading a Nef file and examining the contents.

### The NEF Specification

This along with the examples, NEF validation dictionary and development notes drawn from the (github site defining the standard) https://github.com/NMRExchangeFormat/NEF are in NEF_SPECIFICATION folder

### The NEF Parsing Library

The CCNP NEF parsing package is in the src diectory in the package amd is described in the file NEF_PARSING.md in this directory
